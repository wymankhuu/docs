## Adding Starter Inputs

Starter Inputs allow you to collect information from users before they begin interacting with your app. For the full guide on adding and configuring different types of starter inputs, see our detailed feature documentation.

<Card title="Richer Starter Inputs — Full Guide" icon="arrow-right" href="/features/Richer Starter Inputs">
  Learn about all starter input types (Short Text, Long Text, Dropdown, Checkbox, File Upload), configuration options, best practices, and FAQs.
</Card>

## Quick Overview

Starter inputs help you:
- **Collect key information upfront** before users start interacting with your app
- **Structure conversations** by gathering context like names, grade levels, or preferences
- **Enhance personalization** by using input data to customize the AI's responses

<Tip>
  Remember to preview your app after adding starter inputs to make sure they work as expected!
</Tip>
