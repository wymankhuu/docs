## <Icon icon="bell-on" size="24" className="inline-block align-text-bottom" /> What is this feature?

The Doc Editor is a writing panel built into Playlab apps that lets you create documents without leaving the platform. You can type directly into the editor, use the AI chat to generate content, or combine both approaches — then export your finished document as a DOCX file.

The Doc Editor supports rich formatting including headings, bold, italic, bullet points, numbered lists, tables, and checkboxes — giving you everything you need to produce polished, structured documents from your AI conversations.

## <Icon icon="lightbulb" size="24" className="inline-block align-text-bottom" /> Why is this useful?

This feature bridges the gap between AI conversation and document creation:

* **No more copy-paste:** Generate and refine documents right where the conversation happens
* **AI-assisted writing:** Ask the AI to draft, edit, or restructure your content through the chat
* **Structured output:** Create formatted tables, lists, and layouts directly in the editor
* **Export ready:** Download your document as a DOCX file when you're done — ready to share, print, or edit further

## <Icon icon="thought-bubble" size="24" className="inline-block align-text-bottom" /> How do I use the Doc Editor?

<Steps>
  <Step title="Open a Playlab app with Doc Editor enabled">
    Navigate to a Playlab app that has the Doc Editor tool turned on. You can try it out with the [Doc Editor Preview app](https://www.playlab.ai/project/cmkyeclje43k1ow0u6sim0p25/cml83ez9b53nsme0u937o9jaa).
  </Step>
  <Step title="Open the Doc Editor panel">
    Click the **Doc Editor icon** in the chat window. This opens a blank document panel alongside your conversation.
  </Step>
  <Step title="Create your content">
    You have two options:
    - **Type directly** into the document panel to write and format content yourself
    - **Use the AI chat** to generate content — ask for a lesson plan, table, schedule, or any structured document and it will appear in the editor
  </Step>
  <Step title="Edit and refine">
    Make changes directly in the editor, or ask the AI to modify specific parts of your document through the chat interface. You can format text, add tables, and restructure content as needed.
  </Step>
  <Step title="Export your document">
    When you're satisfied with your document, export it as a DOCX file to share, print, or continue editing in another application.
  </Step>
</Steps>

## <Icon icon="star" size="24" className="inline-block align-text-bottom" /> Tips and Best Practices

<CardGroup cols={2}>
  <Card title="Clear and Start Fresh" icon="eraser">
    Type **"clear"** in the chat to reset your document and start over. You can also start a new session for a completely fresh workspace.
  </Card>

  <Card title="Use Tables" icon="table">
    The Doc Editor supports table creation. Ask the AI to organize information into tables for cleaner, more structured documents — great for lesson plans, schedules, and comparison charts.
  </Card>

  <Card title="Iterate with AI" icon="comments">
    Don't try to get it perfect in one go. Start with a first draft from the AI, then ask follow-up questions to refine specific sections, add detail, or change the tone.
  </Card>

  <Card title="Be Specific in Requests" icon="bullseye">
    The more specific your request, the better the output. Instead of "make a lesson plan," try "create a 5th grade math lesson plan on fractions with a table showing common misconceptions and strategies."
  </Card>
</CardGroup>

## <Icon icon="triangle-exclamation" size="24" className="inline-block align-text-bottom" /> Known Limitations

<Warning>
The Doc Editor is actively being developed. The following limitations are known and being addressed:
</Warning>

* No color or text highlighting support
* Cannot change fonts
* Links do not work in the document editor (external links or breadcrumb links)
* Strikethrough is not supported
* No math LaTeX display support
* Checkboxes work in the editor but do not carry over to DOCX export
* Cannot display inline images or videos without a direct URL
* Some messages may not trigger live doc editing — if this happens, try re-entering your request

## <Icon icon="circle-question" size="24" className="inline-block align-text-bottom" /> Frequently Asked Questions

<Accordion title="How do I enable the Doc Editor in my app?">
  The Doc Editor is available as a tool that can be enabled in the app builder. Toggle the Doc Creator tool on in your app's settings, and the Doc Editor icon will appear in the chat window for users.
</Accordion>

<Accordion title="Can I edit the document after the AI generates it?">
  Yes! The document is fully editable. You can make changes directly in the editor panel — add text, adjust formatting, rearrange content, or delete sections. You can also ask the AI to make specific changes through the chat.
</Accordion>

<Accordion title="What file format does the export use?">
  Documents export as DOCX files, which are compatible with Microsoft Word, Google Docs, and most other word processing applications.
</Accordion>

<Accordion title="Can I paste content into the Doc Editor?">
  Yes. You can paste text into the document panel and then use the AI chat to help you edit, reformat, or expand on the pasted content.
</Accordion>

<Accordion title="Why isn't the Doc Editor icon appearing in my chat?">
  Make sure the Doc Creator tool is enabled in your app's builder settings. In some cases — particularly with Python-based or data analysis apps — the icon may not appear. Try toggling the tool off and on again.
</Accordion>
